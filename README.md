# T-shirt-store
T-shirt store web application :- It is e- commerce website in which user can buy tshirts and buy via any card like visa, maestro ,american etc.
If there is any new user want to buy then firstly he needs to signup then he able to signin and can buy products.
If any admin signin then page redirects to admin area where there are some functions like create category, create products . manage products/category.
